# HospitalManagementSystem
This is simple web application for hospital management system built in django

**#Tech Stack:**
- Python
- Django
- HTML
- CSS
- Javascript
- Bootstrap 
- Sqlite 


**#Features:**

** For Admin:**
 - Login 
 - Logout
 - Add
 - View 
 - Delete
 - Search
 
** For Users:**
 - View Home Page
 - Contact form 
 - View Services
